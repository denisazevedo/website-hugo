## Cloning

```sh
git clone --recursive git@github.com:denisazevedo/website-hugo.git
```

## Installing a new theme

```sh
git submodule add git@github.com:eddiewebb/hugo-resume.git themes/hugo-resume
```

## Running

```sh
hugo serve
```
